Browser

If you’re reading this, you’re using a type of software called a browser, short for web browser. Browsers are used to access and view websites. The most common ones are Chrome, Firefox, Safari, Edge and Internet Explorer, which either are pre-installed or downloadable for your computer and mobile phone.

This is how a browser works:

  • you type in the address you want to access, let’s say youtube.com
  • the browser fetches all resources, such as HTML, CSS, fonts and scripts
  • the browser determines how to render the site, and voilá, you can watch those cat videos!
